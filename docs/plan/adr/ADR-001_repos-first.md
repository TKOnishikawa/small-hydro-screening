# ADR-001: REPOSをマスタとして優先利用する

## ステータス
Accepted (2026-05-05)

## 結論
環境省REPOS が提供する小水力ポテンシャルデータを**一次情報源**とし、未カバー領域のみ標高API＋水文DBで自力算出する。

## 判断理由
1. **計算済みデータの活用**: REPOSは国費で整備された落差・流量・出力の事前計算結果。再計算する合理性が薄い
2. **網羅性**: 全国の主要河川をカバー
3. **形式の標準化**: Shapefile / CSV で提供。GeoPandas で即読込可能

## トレードオフ
- 更新頻度がREPOS側に依存（不定期）
- 細かいパラメータ調整（流量の信頼水準等）はREPOSの前提に縛られる
- → 自力計算オプションを Phase 1 以降で残す
