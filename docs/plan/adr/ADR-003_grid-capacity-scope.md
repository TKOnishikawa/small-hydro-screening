# ADR-003: 系統空き容量データの対象事業者

## ステータス
Proposed (2026-05-05)

## 結論（暫定）
**Phase 3 では1事業者に絞ってPoC**。全10事業者の網羅は後回し。

## 判断理由
- 各一般送配電事業者でデータ公開形式が異なる（CSV / PDF / 地図画像 / 動的Webアプリ）
- 全形式に対応するパーサを書くより、1社で運用回す方が学習効果大
- 対象事業者は Phase 0-2 の REPOSスクリーニング結果から決定可能（候補が集中するエリアの担当事業者を選ぶ）

## 候補
| 事業者 | カバー地域 | 備考 |
|-------|----------|-----|
| 中部電力PG | 岐阜・愛知・静岡・長野・三重 | 山間部の小水力候補多 |
| 関西電力送配電 | 関西2府4県 | 紀伊山地など候補あり |

## 未決事項
- 第一弾の対象事業者は Phase 3 着手時に確定
- 公開データ形式が機械可読でない場合（PDFのみ等）の取扱い
