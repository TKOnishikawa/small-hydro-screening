# 月次更新手順

## 頻度
毎月1日

## 手順
1. REPOSサイトで更新確認（更新あれば対象エリアの最新版をDL）
2. 更新あれば `data/raw/repos/` を上書き（Gitには載せない）
3. `make fetch` で水文DBの最新月分を追加取得（Phase 2 以降）
4. `make screen` で再スクリーニング
5. `output/reports/YYYY-MM/` に結果アーカイブ

## 注意点
- **標高APIは原則キャッシュ利用**。地形は基本変わらない
  - ただし大規模工事・地震等で地形変動があれば該当エリアのみ再取得
- 系統空き容量は事業者の更新サイクルに合わせて取得
- 異常な件数増減（前月比 ±50% 超）が出た場合はアラート → 原因調査
