# 全体仕様: 小水力発電ポテンシャル机上スクリーニング

## 目的
公開データのみを使い、現地調査前の段階で「事業性のある小水力発電候補地」を自動で絞り込む。

## スコープ
- **IN**: 落差・流量・系統空き容量の机上評価、候補地ランキング出力
- **OUT**: 水利権交渉、保安林申請、現地測量、詳細設計

## 完了の定義
1コマンド (`make screen`) で対象エリアの候補地ランキングが CSV / GeoJSON で出力される。

## アーキテクチャ

```
[REPOS]            ─┐
[国土地理院標高API]  ┼→ [統合層] → [スコアリング] → [候補リスト CSV/GeoJSON]
[国交省 水文DB]      ┤                                         ↓
[系統空き容量]      ─┘                                  [QGIS可視化]
```

## 評価式
理論水力出力 (kW) ≈ 9.81 × Q (m³/s) × H (m) × η

- Q: 使用流量
- H: 有効落差
- η: 総合効率（暫定 0.7）
- 詳細: [02_scoring.md](02_scoring.md)
