# スコアリング仕様

## 理論水力出力
P (kW) = ρ × g × Q × H × η / 1000

- ρ: 水の密度 1000 kg/m³
- g: 重力加速度 9.81 m/s²
- Q: 使用流量 m³/s
- H: 有効落差 m
- η: 総合効率（水車 × 発電機 × 伝達） 暫定 0.7

簡易式: **P (kW) ≈ 9.81 × Q × H × 0.7**

## 流量分位の使い分け

| 名称 | 維持日数（年） | 用途 |
|------|--------------|-----|
| 豊水流量 | 95日以上 | 豊水期想定 |
| 平水流量 | 185日以上 | 平均的稼働想定 |
| 低水流量 | 275日以上 | 参考 |
| **渇水流量** | **355日以上** | **設計の保守的下限（優先採用）** |

## 候補スコア（暫定式）
```
score = (P_at_drought × 0.5) + (P_at_normal × 0.3) + (grid_capacity_factor × 0.2)
```
- `grid_capacity_factor`: 系統空き容量。1.0=空きあり / 0.0=空きなし
- 重み係数は Phase 3 完成後に実データで調整

## 除外条件（ハードフィルタ）
- 想定出力 < `MIN_OUTPUT_KW`（環境変数、デフォルト 10kW）
- 落差 < `MIN_HEAD_M`（環境変数、デフォルト 2m）
- 国立公園・国定公園内
- 保安林内
